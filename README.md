# AMZExpand Landing Page

A responsive landing page recreation built as part of a front-end assignment for AMZExpand.

## Live Demo

https://amzexpand.netlify.app/

## Features

* Responsive design for Desktop, Tablet, and Mobile
* Modern landing page UI
* Clean and structured HTML, CSS, and JavaScript
* Reusable components and sections
* Smooth scrolling and interactive elements
* Mobile-friendly layout

## Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Project Goal

The objective of this project was to recreate the provided Figma design as closely as possible while maintaining:

* Design accuracy
* Responsive layout
* Clean code structure
* Good user experience
* Front-end best practices
